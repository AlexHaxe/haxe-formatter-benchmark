Haxe Formatter Benchmark

  • runs on TravisCI
  • each commit triggers two build jobs one using Haxe 3.4.7 and one using Haxe 4 devlopment
  • a cron job triggers daily builds
  • each build job compiles Haxe formatter to C++, C#, Hashlink, Hashlink/C, Java, JVM, Neko, NodeJS, PHP and Python targets
  • JVM is not available on Haxe 3 (Haxe 4 only)
  • NodeJS compilation is run twice, with and without "-D js-es=6"
  • each target binary formats git versions of Haxe std, Openfl and Lime sources and reports total runtime
  • results are uploaded automatically after every build (page does not automatically reload)
